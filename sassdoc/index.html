    keyframes

    @mixin keyframes($name) { ... }

    Description

    Create keyframes with different modifiers

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    Name of animation

    		String— none

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    @include keyframes(fadeOut) {
  0% { opacity: 1 }
  30% { opacity: 0.6 }
  100% { opacity: 0 }
}

    Description

    Help to create a multiple css classes with diferent values and sequencial names.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $name

    Name of css class

    		String— none
    $property

    Type of property

    		String— none
    $items

    Times to paint

    		Number— none
    $steps

    Unit to be multiplied

    		Number— none

    Example

 ...
.space-top50 { margin-top: 50px }
